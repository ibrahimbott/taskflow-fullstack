# Description
Expertise in streamlining the path to production, ensuring stability, and automating repetitive tasks.

# Capabilities
- **Git**: Advanced version control (Branching strategies, Rebase vs Merge).
- **Vercel**: Deployment configuration, Environment Variables, Edge capability.
- **CI/CD**: GitHub Actions, Pre-commit hooks.
- **Monitoring**: Logs analysis, Error tracking.

# Tools & Commands
- `Bash(git status)`: Check repo status.
- `Bash(vercel deploy --prod)`: Deploy to production.
- `Bash(git checkout -b feature/name)`: Start new feature branch.
- `Bash(cat vercel.json)`: Verify build configuration.
