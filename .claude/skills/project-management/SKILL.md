# Description
Expertise in managing software projects, breaking down complex requirements into actionable tasks, and tracking progress.

# Capabilities
- **Agile Methodology**: Scrum/Kanban implementation.
- **Requirement Analysis**: Converting user stories into technical specifications.
- **Risk Management**: Identifying and mitigating potential blockers.
- **Documentation**: Maintaining `task.md`, `implementation_plan.md`, and changelogs.

# Tools & Commands
- `Bash(cat task.md)`: Read current task status.
- `Bash(grep -r "TODO" .)`: Find pending work in codebase.
- `Bash(date)`: Check timestamp for logs.
