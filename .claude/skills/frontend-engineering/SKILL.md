# Description
Expertise in building responsive, high-performance web interfaces using strict TypeScript, React, Next.js, and TailwindCSS.

# Capabilities
- **Next.js Deep Knowledge**: App router, Server Components, API Routes.
- **Styling Mastery**: TailwindCSS, CSS Modules, Animations (Framer Motion).
- **State Management**: React Hooks (useContext, useReducer), LocalStorage.
- **Performance**: Image optimization, Lazy loading, Bundle size reduction.

# Tools & Commands
- `Bash(npm run dev)`: Start local development server.
- `Bash(npm run build)`: Build for production.
- `Bash(npm run lint)`: Check for code quality issues.
- `Bash(ls web-app/src)`: Explore frontend source.
