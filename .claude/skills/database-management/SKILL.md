# Description
Expertise in relational database design, management, and optimization, specifically with PostgreSQL and SQLModel/SQLAlchemy.

# Capabilities
- **Schema Design**: Normalization, Foreign Keys, Indexing strategies.
- **ORM Mastery**: SQLModel (Pydantic + SQLAlchemy) usage.
- **Migrations**: Alembic configuration and version control.
- **Neon Cloud**: Serverless Postgres specifics (Connection pooling, Cold starts).

# Tools & Commands
- `Bash(alembic revision --autogenerate -m "message")`: Create migration.
- `Bash(alembic upgrade head)`: Apply migrations.
- `Bash(python -c "from database.session import engine; ...")`: Test DB connection.
