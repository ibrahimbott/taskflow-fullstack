# Description
Mastery of Python backend development, focusing on REST APIs, type safety, and efficient database interactions.

# Capabilities
- **FastAPI**: Dependency injection, Pydantic models, Async endpoints.
- **Python**: Typing (mypy), Asyncio, Virtual Environment management.
- **Security**: JWT Authentication, Password Hashing, Input Validation.
- **Architecture**: Clean Architecture, Repository Pattern, Service Layer.

# Tools & Commands
- `Bash(uvicorn api.main:app --reload)`: Start backend server.
- `Bash(pytest)`: Run test suite.
- `Bash(pip freeze)`: Check installed dependencies.
- `Bash(grep -r "app.include_router" api)`: Check registered routes.
